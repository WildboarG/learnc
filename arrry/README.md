4. 数组名

   ​	数字名是表示地址的常量，也是数组的起始位置。

5. 数组越界

   ​		越界部分可以赋值 a[i]=*(a+i)   指针偏移合法 不属于数组了

   

常见题目：

* 求fibonacci数列的前10项，并在数组中逆序存放

* 进制转换
* 删除法求质数
